Why it matters

Your docs have a new audience: AI is reading them now

The invisible audience

AI coding agents are reading your API docs millions of times a day. If your docs aren't optimized, you're invisible to the fastest-growing segment of your user base.

Lighthouse for AI agents

Agent Score is the first industry benchmark for AI-agent readiness. Think Lighthouse, but for how effectively AI agents can discover, parse, and use your documentation.

Agent readiness = moat

API platforms with higher agent readability are seeing outsized adoption. Agent readiness is the new SEO. 73% of top API docs still lack an llms.txt.

Open source, no black boxes

Built on the open-source Agent-Friendly Docs Spec. No gatekeeping, no black boxes. Every check is transparent and community-driven.
