We built a Lighthouse Score for the age of AI agents.

When Dachary Carey published the Agent-Friendly Documentation Spec, the research was there. What was missing was a way to make it stick. Lighthouse worked because it gave developers a number to chase. We think the same is true here.

Agent Score runs 22 automated checks across your docs and returns a grade. It's built on Dachary's open spec, fully open source, and takes under 2 minutes to run.

If AI agents can't use your docs, developers using Cursor, Claude, and ChatGPT won't reach your API. They'll reach for a competitor whose docs work.
